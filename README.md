# Installation

### 1. Database config

Create account and database in localhost mysql server, then change config in `config/db.php`
```
return [
    'class' => 'yii\db\Connection',
    'dsn' => 'mysql:host=localhost;dbname=yii2lucene',
    'username' => 'root',
    'password' => 'root',
    'charset' => 'utf8',
];
```
